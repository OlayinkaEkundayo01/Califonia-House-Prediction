# California-House-Prediction
The project aims at building a model to predict median house values in California based on the longitude and latitude in the city

The US Census Bureau released the California Census Data, which includes ten different metrics for each Califonia block group, including the population, median income, median home price, and so on. The dataset attempts to define the functional and nonfunctional requirements for the project and is also used as an input for project scoping.

# Data Source link: https://www.kaggle.com/datasets/shibumohapatra/house-price

# Methodology
 1.Build a model  to predict median house values in California based on Longitude and latitude features from the dataset.
 2.Train the model to learn from the data to predict the median housing price given the longitude and latitude in Califonia.
 3 Predict housing prices based on latitude and longitude location and plot the regression chart for it.
